# Day 1 Answers

## 1. What problem does Salesforce solve?

Salesforce helps businesses manage customer information, sales, marketing, and support services in one place.

## 2. What is CRM?

CRM stands for Customer Relationship Management.

It helps companies manage interactions with customers and improve customer service.

## 3. What is an Object in Salesforce? Give one example.

An Object in Salesforce is used to store data.

Example:
Contact Object stores customer details like name, phone number, and email.

## 4. Difference between Salesforce Admin and Developer

| Salesforce Admin | Salesforce Developer |
|---|---|
| Manages users and settings | Builds custom applications |
| Uses configuration tools | Writes code |
| Creates reports and dashboards | Develops integrations |

## 5. Suggest one real-world application that can be built using Salesforce.

Hospital Management System

It can manage patient records, appointments, billing, and doctor information.
